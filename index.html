Lyricity

Check it out here!!

This is Lyricity. My team and I created a web lyrics app that allows a user to search for a song that they want to know the lyrics to. If the song is in our database it will generate the lyrics and album cover if available. The user can then favorite that song to save it to their favorites. If the song is not in our database the user will be greeted with an inspiring Kanye quote.
